Data Quality Issues

IMD 39.6% missing — Labels "1 (most deprived)" broke as.integer() in R. Only quintiles 2–4 visible. FIXED
Immunosuppression (medication) = 0% — DMD drug codes queried against SNOMED field; zero matches. REMOVED
Registration date fixed to 2017-01-01 — All patients required registration before study start, not before their index date. FIXED

Methodological Issues

Antibiotic window overlap — Prior-month and prior-year windows overlapped; index date counted as exposure. REDESIGNED
AMR ±30d window too wide — May capture unrelated episodes. NARROWED TO ±14d
Missing risk factors vs CPRD — Only 27 of 60 conditions from van Staa 2024. EXPANDED TO 39
No primary care BSI — Community-managed episodes missed. COMBINED DATASET ADDED
No incidence rates — Raw counts only, no population denominators. MEASURES ADDED
No control cohort — Cannot answer "who develops BSI?" CONTROLS ADDED
✅ Changes Implemented for Round 2

Data Fixes

IMD → built-in .imd_quintile — Integer 1–5, no label parsing. Expect 5 quintiles with minimal missingness.
Registration → rolling — Now requires ≥1yr before each patient's own index date (not study start).
Immunosuppression (medication) deleted — Diagnosis-based variable retained (9.8%).

Methodological Improvements

Antibiotic windows redesigned — 4 non-overlapping variables: abx_6w, abx_prior_year_cat, abx_classes_6w, abx_classes_prior_year. Index date excluded.
AMR window ±14 days — Matches blood culture turnaround + GP coding delay.
+12 comorbidities — Frailty (eFI), housebound, PVD, VTE, organ transplant, IBD, RA, Parkinson's, HIV, osteoporosis, hypothyroidism.
12 antibiotic classes — 3,663 DMD codes classified: penicillins, cephalosporins, macrolides, fluoroquinolones, tetracyclines, aminoglycosides, carbapenems, glycopeptides, nitroimidazoles, sulfonamides/trimethoprim, nitrofurantoin, other.

New Datasets

Combined dataset — Hospital ICD-10 + primary care SNOMED BSI. Source classification: hospital_only / primary_care_only / both.
Measures framework — Monthly BSI rates per registered population. 7 measures: overall, resistant, by IMD, by region, by age, by sex. 108 monthly intervals.
Control cohort — No-BSI population pool. Same 39 comorbidities extracted. Ready for 1:6 osmatching.
